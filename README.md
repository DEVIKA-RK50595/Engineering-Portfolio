Live Deployment Link: **[engineering-portfolio-green.vercel.app](https://engineering-portfolio-green.vercel.app)**

## 🛠️ Tech Stack & Architecture

* **Framework:** Next.js (App Router, Client Component Optimization)
* **Styling:** Tailwind CSS (Custom ambient layer blends, micro-interactions, dark aesthetic)
* **Icons:** Lucide React & High-Fidelity Custom SVGs (for zero-dependency asset delivery)
* **Deployment:** Vercel Global Edge Network (CI/CD integrated via GitHub)
